Classes

This project provides a curated set of CSS classes designed to streamline the styling of your website. These classes are generated through custom-built Sass mixins specifically tailored for this project, offering a solid foundation that can be easily expanded to include additional styles as needed. The included classes cover fundamental styling needs, serving as a practical starting point for building an application. Additionally, a CSS purging process is applied to ensure the final CSS file remains lightweight, containing only the classes actively used within the project.
